< a >
The < a > tag is used to Link elements.
< abbr >
The < abbr > tag is used to show abbreviations.
< b >
The < b > tag is used to make text bold.
< basefont >
The < basefont > tag is used to set default font properties.
< bdo >
tag is used to reverse the direction of text.
< big >
The < big > tag increases font size.
< br >
The < br > tag is used to create a line break in a block of text.
< cite >
The < cite > tag is used to show citations in text.
< code >
The < code > tag identifies computer code.
< dfn >
The < dfn > tag defines a term.
< dir >
The < dir > tag is used to contain you < li > tags, and other list tags.
< div >
The < div > tag is used as a block level container.
< dl >
The < dl > tag is used to contain definition terms, such as dd and dt.
< em >
The < em > tag adds emphasis to text.
< fieldset >
The < fieldset > tag is used to form a field grouping container.
< font >
The < font > tag sets font values
< form >
The < form > tag is used to set the start and end point of a form.
Heading Tags
Heading tags are used to place heading on your webpage and number from 1-6 ( example < h1 >< /h1 >) decrementing as the number increases.
< i >
The < i > tag italicizes text.
< img >
The < img > tag imbeds an image into a document.
< input >
The < input > element allows users to input data.
< ol >
The < ol > tag creates an ordered list.
< p >
The < p > tag creates a paragraph of text.
< ul >
The < ul > tag creates an unordered list.
< u >
The < u > tag underlines text.
< strong >
The < strong > tag puts a strong emphasis on text.
